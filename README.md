### Exercise 1.2

ArrayList

* Fast random access
* Best for frequent reads

LinkedList

* Efficient insertion and deletion
* Slower random access

HashSet

* Prevents duplicates
* Fast lookup

TreeSet

* Maintains sorted order
* Slightly slower than HashSet

Recommended Collection:
ArrayList is the most suitable choice for recently viewed products because it provides fast retrieval and iteration.
